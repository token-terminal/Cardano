# Grant Milestone 1 – Architecture & Schema Documentation

## Overview

This repository contains the public submission for **Milestone 1**, fulfilling the requirement:

> “Architecture and schema documentation completed and stored in a public GitHub repository.”
> 

> “Includes diagrams or structured descriptions of data flow and table definitions.”
> 

The documentation describes the system architecture and data schema as required by the grant milestone and is provided in a publicly accessible format for review.

---

## Deliverable

**Architecture & Schema Documentation (PDF)**

- Location: `Milestone 1/architecture-schema-submission.pdf`

The document provides comprehensive architecture and schema documentation, including:

- A concise ELT architecture overview, describing the data flow from the RPC source through the extraction and loading processes into the Token Terminal data warehouse
- Clearly defined staging table schemas for blocks and transactions
- A description of the tools and infrastructure used for data extraction and loading
- Structured data-flow documentation, covering:
    - RPC configuration and setup
    - ELT pipeline design
    - Data ingestion and processing details
- Supporting evidence, including screenshots of pipeline logs and query results demonstrating successful ingestion and validation

This documentation satisfies the milestone requirement by fully specifying the system architecture, data schemas, and operational data flows in a publicly accessible repository.
